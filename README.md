# POS Automation Testing
Automation testing practice project for a Point of Sale (POS) web application using Playwright and TypeScript.

## About Project
This project was created to practice end-to-end automation testing using Playwright with TypeScript. The automation scripts follow the Page Object Model (POM) design pattern to improve readability, maintainability, and scalability.

## Tech Stack
- Playwright
- TypeScript
- Node.js
- Visual Studio Code
